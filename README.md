# trendchart

Small, simple [trendchart](https://github.com/WebLogin/trendchart-elements) for R.

## Installation

``` r
# install.packages("remotes")
remotes::install_github("the-y-company/trendchart")
```

## Example

``` r
trendchart::trend_line(sample(1:100))
```
